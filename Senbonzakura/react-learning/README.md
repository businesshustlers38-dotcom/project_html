# React Learning Project

Detta är ett enkelt React-projekt för att lära sig komponenter, props, state och events.

## Kom igång

1. Installera dependencies: `npm install`
2. Starta appen: `npm start`
3. Öppna [http://localhost:3000](http://localhost:3000) i din browser.

## Vad du lär dig

- **Komponenter**: Funktionella komponenter som `Welcome`, `Counter`, `ItemList`.
- **Props**: Skicka data mellan komponenter, t.ex. `name` i `Welcome`.
- **State**: Använd `useState` för att hantera tillstånd, t.ex. räknare i `Counter`.
- **Events**: Hantera händelser som `onClick` för knappar.

Experimentera genom att ändra koden och se resultaten!
